## 🎨 设计特色

### 摄影主题配色方案
```css
/* 背景层次 */
--color-dark-bg: #0a0a0a;           /* 深黑背景 - 暗房感觉 */
--color-surface-primary: #1a1a1a;   /* 主要表面 - 相机机身 */
--color-surface-secondary: #2a2a2a; /* 次要表面 - 镜头环 */
--color-surface-accent: #1e1e1e;    /* 强调表面 - 取景器 */

/* 摄影主题强调色 */
--color-accent-gold: #d4af37;       /* 金色 - 专业品质 */
--color-accent-silver: #c0c0c0;     /* 银色 - 金属质感 */
--color-accent-amber: #ffb000;      /* 琥珀色 - 暖光 */
--color-accent-cyan: #00bcd4;       /* 青色 - 冷光 */
--color-accent-red: #e53e3e;        /* 红色 - 录制指示 */
```

### 视觉层次设计
| 层级 | 背景样式 | 边框样式 | 标题颜色 | 用途 |
|------|---------|---------|---------|------|
| 主区块 | 纯色背景 + 卡片阴影 | 细边框 | 金色 | 顶级分类 |
| 子区块 | 渐变背景 + 元素阴影 | 左边框 | 青色 | 二级分类 |
| 器材容器 | 深色背景 + 边框阴影 | 全边框 | 琥珀色 | 器材展示 |
| 器材项目 | 渐变背景 | 细边框 | 红色 | 具体类别 |
| 特殊项目 | 双色渐变 + 特殊阴影 | 金色边框 | 金色 | 重要信息 |

## 🔧 技术实现亮点

### 1. 避免样式嵌套
通过差异化设计避免相同样式的卡片嵌套：
- 不同层级使用不同的背景样式
- 渐变、纯色、深色背景的组合
- 边框样式的差异化处理

### 2. 响应式布局修复
修正器材部分的响应式问题：
```diff
- "grid grid-cols-2 lg:grid-cols-1"  // 错误：宽屏1列，窄屏2列
+ "grid grid-cols-1 lg:grid-cols-2"  // 正确：宽屏2列，窄屏1列
```

### 3. 交互体验优化
- 标签悬停效果：颜色反转 + 平滑过渡
- 渐变背景：营造立体感
- 阴影系统：增强层次感
