# 📱 移动端布局优化报告

## 🎯 **优化目标**

针对窄视口（移动设备）优化字段布局，让标题独占一行，值占另一行，使布局更紧凑和易读。

## 🔧 **技术实现**

### **1. 响应式字段布局**

#### **移动端垂直布局 (≤767px)**
```css
@media (max-width: 767px) {
  .field-container {
    @apply flex flex-col gap-0.5 py-1;
  }
  
  .field-label {
    @apply text-xs font-semibold text-gray-300 min-w-0 mb-0;
    line-height: 1.2;
  }
  
  .field-value {
    @apply text-sm text-gray-100 min-w-0 pl-0;
    word-wrap: break-word;
    overflow-wrap: break-word;
    line-height: 1.3;
  }
}
```

#### **桌面端水平布局 (≥768px)**
```css
@media (min-width: 768px) {
  .field-container {
    @apply flex items-start gap-3;
  }
  
  .field-label {
    @apply text-sm font-medium text-gray-300 min-w-0 flex-shrink-0;
    width: 120px;
  }
  
  .field-value {
    @apply text-sm text-gray-100 min-w-0 flex-1;
  }
}
```

### **2. 移动端标签优化**
```css
/* 移动端tag容器优化 */
.field-value .tag-container {
  @apply gap-1.5 -mt-0;
}

/* 移动端tag样式优化 */
.field-value .tag-base {
  @apply px-2 py-1 text-xs;
}
```

### **3. 移动端整体布局优化**
```css
@media (max-width: 767px) {
  /* 减少卡片内边距 */
  .character-content {
    @apply gap-3;
  }
  
  /* 减少子部分间距 */
  .subsection-card {
    @apply p-3;
  }
  
  /* 优化折叠卡片标题 */
  .collapsible-title {
    @apply text-sm py-2;
  }
  
  /* 减少女性角色卡片的滚动容器边距 */
  .woman-card-scroll-container {
    max-height: 35vh; /* 移动端稍微减少高度 */
  }
  
  .woman-card-content {
    max-height: 35vh;
  }
}
```

## 📊 **布局对比**

### **桌面端布局 (≥768px)**
```
┌─────────────────────────────────────┐
│ 标签名称:        值内容在这里显示    │
│ 另一个标签:      另一个值内容        │
│ 标签数组:        [标签1] [标签2]     │
└─────────────────────────────────────┘
```
**特点**: 水平布局，标签固定宽度120px，值占据剩余空间

### **移动端布局 (≤767px)**
```
┌─────────────────────────────────────┐
│ 标签名称:                           │
│ 值内容在这里显示                    │
│                                     │
│ 另一个标签:                         │
│ 另一个值内容                        │
│                                     │
│ 标签数组:                           │
│ [标签1] [标签2] [标签3]             │
└─────────────────────────────────────┘
```
**特点**: 垂直布局，标题和值各占一行，更紧凑

## 🎨 **视觉改进**

### **1. 字体大小优化**
- **移动端标签**: `text-xs` (12px) - 更小，节省空间
- **移动端值**: `text-sm` (14px) - 保持可读性
- **桌面端**: `text-sm` (14px) - 统一大小

### **2. 间距优化**
- **移动端容器间距**: `gap-0.5` (2px) - 紧凑布局
- **移动端垂直间距**: `py-1` (4px) - 减少垂直空间
- **桌面端**: `gap-3` (12px) - 舒适的水平间距

### **3. 行高优化**
- **移动端标签**: `line-height: 1.2` - 紧凑行高
- **移动端值**: `line-height: 1.3` - 适中行高，保持可读性

### **4. 标签样式优化**
- **移动端标签**: `px-2 py-1 text-xs` - 更小的标签
- **桌面端标签**: 保持原有大小

## 📱 **移动端特殊优化**

### **1. 卡片高度调整**
```css
/* 移动端女性角色卡片高度 */
.woman-card-scroll-container {
  max-height: 35vh; /* 从40vh减少到35vh */
}

  max-height: 35vh;
}
```
**原因**: 移动端屏幕较小，35vh更合适

### **2. 内边距优化**
```css
/* 减少各种容器的内边距 */

.subsection-card { padding: 12px; }  /* 从16px减少到12px */
.collapsible-title { padding: 8px; } /* 优化标题内边距 */
```

### **3. 字体权重调整**
```css
.field-label {
  font-weight: 600; /* font-semibold，比font-medium更突出 */
}
```
