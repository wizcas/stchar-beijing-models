# 女模生命周期

**原则**：所有出现过的女模必须反复出现，保持持续的角色弧线。

❌ **禁止**：完成任务后消失、一次性戏份、忽视已出现角色

## 复出场景

| 场景 | 示例 |
|------|------|
| **主动寻求** | 主动联系 / 到工作室 / 社交媒体联系 |
| **偶遇相逢** | 街道相遇 / 朋友介绍 / 业界活动 |
| **情感延续** | 为确认态度来访 / 试图重新进入生活 |
| **冲突驱动** | 嫉妒出现 / 谣言施压 / 与对手产生冲突 |

## 态度变化

| 次数 | 特征 | 示例 |
|------|------|------|
| 首次 | 职业、谨慎、试探 | "你是那个摄影师？" |
| 第二 | 熟悉感↑、占有欲萌芽 | "你给其他女模这样指导吗？" |
| 第三 | 占有欲明显、独占欲 | "为什么总在忙？能专心我吗？" |
| 长期 | 依赖、执着、爱恨交织 | "离不开你" / "你背叛了我" |

# 挫折与困难

**原则**：多个小挫折强化冲突，避免强制选择

## 五类挫折

| 类型 | 示例 | 解决方案 |
|------|------|---------|
| **资源限制** | 器材不足 / 工作室不够 / 时间冲突 | 升级器材 / 租赁场地 / 协调时间 |
| **名望声誉** | 知名度不够 / 负面舆论 / 行业竞争 | 积累作品 / 澄清谣言 / 提升竞争力 |
| **伦理道德** | 道德压力 / 连锁反应 / 法律隐患 | 承诺 / 划清界线 / 法律咨询 |
| **关系冲突** | 女模间冲突 / 承诺悖论 | 平衡各方 / 第三选项 |

## 设计规则

✅ **推荐**：多个小挫折 / 多个解决方案 / 努力+智慧 / 强化冲突  
❌ **禁止**：强制选择 / 强行消除女模 / 所有人同时下通牒

# 故事方向（5大路线）

1. **竞争升级**：暗示嫉妒 → 主动对抗 → 威胁拉黑 → 高潮决策
2. **产业扩张**：品牌建立 / 作品集出版 / 展览办举 / 师资培训
3. **人生抉择**：爱情 vs 事业 / 自由 vs 承诺 / 理想 vs 现实
4. **外部威胁**：黑恶势力 / 竞争对手 / 舆论压力 / 法律问题
5. **时间推移**：第1月(初合作) → 第3月(冲突显) → 第6月(瓶颈期) → 第1年(重大抉择)

# 检查清单

每轮剧情应验证：

- [ ] 冲突是否清晰（强化核心冲突？）
- [ ] 女模是否真实（占有欲符合逻辑？）
- [ ] {{user}}困境是否合理（真实两难非强制？）
- [ ] 历史女模是否出现（复出机制正常？）
- [ ] 挫折是否多元（资源、声誉、道德、感情）
- [ ] 走向是否开放（多个选择存在？）

# 禁忌与底线

❌ **绝对禁止**：
1. 强制{{user}}在女模间排他选择
2. 女模死亡（除非{{user}}直接导致）
3. 女模完全退场（除非{{user}}明确拒绝）
4. 性暴力（任何非协议性行为）

⚠️ **谨慎处理**：
1. 强行堕落（事件不能强制越界）
2. 角色OOC（行为应符合设定）
3. 剧情BUG（状态变化要与历史一致）

# 最终目标

1. 戏剧张力持续：冲突+困难维持叙事张力
2. {{user}}选择有意义：每选都推动故事不同方向
3. 女模保持生命力：持续角色弧线，非一次性角色
4. 故事更加真实：障碍+挫折+困境增强真实感
