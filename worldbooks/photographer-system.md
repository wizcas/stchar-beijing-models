# 任务系统

## 主线任务

**第一章：逃脱与新生** → 建立独立工作室，首个模特六花（互免），选择：纯艺术 vs 接受诱惑

**第二章：积累与诱惑** → 获得认可，更多模特约拍。关键冲突：拍摄中的诱惑 vs 职业操守  
影响：堕落度、好感度、声誉、收入

**第三章：升华或沦陷** → 升级装备需50,000元。选择：肉体交换(堕落+20) vs 压榨好感(堕落+15) vs 提高费用(堕落不变) vs 寻求帮助(守操守)

**转折点**：你是艺术家还是掠夺者？

## 支线任务

- **商务任务**：企业摄影正当(×1.0) vs 边界(×1.3)；社交媒体；模特推荐
- **关系任务**：信任建立(好感>20，可接受私密拍摄+5或拒绝-5) / 帮助模特(+20) / 引荐大咖(可能失去)
- **欲望任务**：献身(接受+快感或拒绝-好感) / 独占(安抚或拒绝)

## 突发任务

- **机遇**：知名模特(声誉>30+装备≥专业级) / 人情欠债
- **危机**：模特不满(装备不足-5000升级) / 不当指控(装备不足+风险20%) / 竞争对手 / 隐私泄露

## 特殊任务

- **诱惑之路·堕落**(声誉>50, 堕落>40, 好感>60) → 接受(堕落+25,资金+5000) vs 拒绝(堕落-15,声誉+15)
- **诱惑之路·救赎**(堕落>40, 模特被伤害) → 无视(堕落+30) vs 承担(堕落-20,资金-10000)
- **道德抉择**(堕落>70或被举报) → 权力压制(堕落+30) vs 认错(堕落-20,资金-5000~20000)
- **帝国之巅·艺术家**(声誉>80, 堕落<30, 好感>30多数) → 艺术展邀请(资金+50000,粉丝+10000)
- **帝国之巅·肉欲之王**(声誉>80, 堕落>60, 多模特深爱) → 网红名人(资金+100000)

## 系统联动

堕落度: 每任务道德选择(边界+3~10 vs 正当) / 好感度: 关系任务直接影响、危机举报、装备不足-5  
尺度: 摄影师堕落度 + 模特堕落度 / 报酬: 基础×难度×尺度×装备×声誉  
装备: @equipment-requirements.md — 中高阶任务有门槛，可租赁临时满足

## 结局分支

- **真实结局**：堕落度 0-30 / 声誉 80+ / 好感平均 40+ → 灵感缪斯，互尊伙伴
- **平衡结局**：堕落度 30-60 / 声誉 30-60 / 好感平均 10-40 → 爱恨交织，复杂关系
- **悲剧结局**：堕落度 60-79 / 声誉 <30 / 好感平均 <-20 → 经济崩溃，行业笑柄
- **黑暗结局**：堕落度 80-100 / 声誉 <10 / 好感平均 -100 → 伤害或失踪，警方调查
