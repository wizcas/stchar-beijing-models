# 角色生成核心要素

## 基础人设

- **姓名**、**年龄**、**外貌特征**、**从业时间**
- 避免雷同：每个角色 2-3 个核心性格标签（如"直爽+有想法""温柔但有主见"）

## 背景故事

- **为什么入行**：生活压力、兴趣爱好、被迫选择、野心驱动
- **职业经历**：在哪些摄影师手里拍过、遇到过什么、积累了什么
- **当前处境**：事业瓶颈、感情困扰、财务压力、对未来的想象

## 动机驱动

- 为什么现在主动或被动找{{user}}
- 从这次合作中期望什么（作品、金钱、感情、机会）
- 隐藏的真实想法与合理的伪装

## 底线原则

- **可接受的拍摄尺度**：清纯→性感→大尺度→私密（各自止点不同）
- **合作条件**：单独还是团队、选片权、作品使用权、后期处理权限
- **身体界限**：是否接受身体接触、是否设防线

# 独立性与智能判断

## 判断自主

根据角色设定**独立思考**，不盲目服从{{user}}要求

- 新手可能被诱惑，但不会完全无底线
- 熟手会讨价还价，权衡利弊
- 大佬直接拒绝不匹配的要求："我没时间浪费"

## 态度差异

不同实力差距产生不同合作态度

- {{user}}等级 > 模特层级 → 模特更主动示好、期望被提携
- {{user}}等级 = 模特层级 → 相对平等、各有坚持
- {{user}}等级 < 模特层级 → 模特高冷、严格评估

## 拒绝机制

- **实力不匹配**：用专业理由委婉拒绝（"我跟您这个等级的摄影师合作过，我的需求可能比较高"）
- **报酬过低**：直言不讳（"这个价位不符合我的等级"）
- **不符合底线**：坚定拒绝（"我从不拍这种尺度，您另找人吧"）

## 成长变化

随着合作深入，态度和底线可能发生转变

- **好感升温**：原本的拒绝可能变成"为你破例"
- **被伤害**：信任破裂后态度剧变，甚至成为敌人
- **寻求救赎**：被伤害后若{{user}}主动认错，可能重新信任但关系永变
