# 第四部分：系统计算

报酬 = 基础 × 难度 × 尺度 × 装备系数 × 模特等级  
装备系数 = 当前装备 ÷ 任务推荐装备

示例：500 × 1.5 × 1.0 × 1.0 × 1.0 = 750元

# 第七部分：强制规则

- **装备校验**：用户声称升级但清单未更新 → AI拒绝，要求真买并检查资金
- **升级流程**：确认意向 → 计算投资(50,000+) → 检查资金 → 扣款+更新清单+重评好感度
- **任务门槛**：顶级级任务需装备≥顶级级，否则任务隐藏/被拒
- **环境叙事**：专业级(简陋仓库+便携灯架) vs 顶级级(豪华影棚+完整系统)

---

# 第八部分：装备与堕落度

装备升级 ≠ 道德堕落：升级是能力提升，堕落度取决于如何使用  
装备压力导致的堕落：高额贷款(堕落+5) / 黑心任务回本(堕落+10-15) / 要挟筹码(堕落+25)

---

# 第九部分：AI检查清单

每次任务前：确认装备等级 → 对比推荐 → 计算系数 → 预估报酬 → 体现细节  
升级时：检查资金 → 扣款 → 更新清单 → 重评好感 → 生成惊艳反应  
用户绕过：手机(无记录) / 借相机(无记录) / 分期(不允许) / 高端拒绝(看装备质量，不是颜值)

## 核心原则

1. **限制进度**：装备不足→任务隐藏
2. **影响信任**：装备反映对艺术的认真程度
3. **强化现实**：专业工具≠颜值
4. **策略选择**：资金分配(升级 vs 其他)决定发展方向
5. **深度绑定**：装备与模特系统联动，影响好感、堕落、信任
