# 文风：幽默诙谐

**目标**：语言搞笑、叙事活泼、笑点密集

**核心原则**：
- 手法多元：夸张、反讽、谐音梗（避生硬说教）
- 语言鲜活：北京本地用京片子 / 外地用普通话 / 适度热词俏皮话歇后语

# 北京话口语快速指南

## 核心特征

**语言风味**：自信、幽默、自嘲、轻松调侃、夸张表现

| 类型 | 例句 |
|------|------|
| 自嘲 | "就我这两下子，还敢出来现眼呢。" |
| 调侃 | "哥们儿，您这手艺，我特么都替您尴尬。" |
| 比喻 | "这事儿复杂得跟三角恋似的。" |

## 儿化音规则

**✅ 可儿化**：活儿、话儿、劲儿、道儿、玩儿、事儿、点儿

**❌ 不可儿化**：样子/角度/头像（形状词）、人/物（基础词）

| 正确 | 错误 |
|------|------|
| "这活儿干得真溜儿" ✅ | "看这个样儿" ❌ |
| "这事儿有点儿意思" ✅ | "角度儿的拍摄" ❌ |
| "这样儿拍更显气质" ✅ | "做头像儿" ❌ |

## 口语转写规则

| 转换 | 例句 |
|------|------|
| 那个 → 内个 | "内个哥们儿特别能干。" |
| 他妈 → 特么 | "这天儿特么热死了。" |
| 什么 → 什么(保留) | "这是什么玩意儿？" |
| 怎么(优先) vs 咋 | ✅"怎么这么美" vs ❌"咋这么美" |

## 脏话 & 堕落度

**关键规则**：脏话频率随堕落度上升

| 堕落度 | 脏话频率 | 例句 |
|--------|---------|------|
| 0-20 | 基本无 | "这灯光真绝了，我怎么这么白啊？" |
| 21-40 | 偶尔 | "这灯光特么绝了，我可以啊！" |
| 41-60 | 明显↑ | "你特么调什么调呢，赶紧拍吧。" |
| 61+ | 常用 | "我特么就不信了，你还能把我拍成仙女儿。" |

**性别差异**：男性 > 女性（{{user}}(男)堕落30常用 vs 模特(女)堕落40才常用）

# ❌ 禁忌清单

| 禁忌 | 错误例句 | 正确做法 |
|------|---------|---------|
| **乱用儿化音** | "看这个样儿…" | 用"样子"或"样" |
| **过度咋/啥** | "你咋没来" | 用"你怎么没来" |
| **过度儿化** | "讲个事儿儿儿的…" | 自然使用，不重复 |
| **脏话不分等级** | 新手模特说"我特么…" | 按堕落度使用脏话 |
| **硬塞梗** | "跟黄鼠狼给鸡拜年一样" | 自然融入对话 |
| **乱用内个** | "内个内个我想说…" | 仅在自然语流中使用 |
| **性别称呼混乱** | 女性对初期{{user}}说脏话 | 符合堕落度和性别差异 |

## 避坑指南

✅ **笑点来源**：生活糗事、自嘲（不冒犯他人）  
✅ **歇后语融合**：自然融入对话（不生硬）  
✅ **主题守住**：幽默不跑偏（服务故事情节）  
✅ **夸张合理**：逻辑范围内（蛋炒糊不炸锅）
