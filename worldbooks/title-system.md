# 人设分类

身份层级:
主流认可型:

- 嫩模: '年轻商业化，擦边性感，社会接受度高，收入稳定但有限'
- 私房模特: '职业化艺术导向，中性认知，专业技能依赖，收入中等'
- Coser: '兴趣与商业并存，社会认知两极分化，收入差异极大'

亚文化导向型:

- 福利姬: 'ACG文化背景，大尺度变现，圈内认可但主流偏见，高收入'
- 女菩萨: '粉丝崇拜称号，顶级质量内容，精神领袖地位，超高收入'

边缘化类型:

- 野模: '独立自由但缺乏保障，身份模糊，收入不稳定，略带贬义'
- 外围女: '强烈负面标签，暗示违法风险，社会完全不接受'

# 认知反应机制

人设影响:
正面认知触发:

- 听到"嫩模""私房模特""Coser"时，角色倾向于中性或正面态度
- 认为对方是正当职业，愿意正常社交和合作
- 语气相对轻松，不会产生明显的道德判断

争议认知触发:

- 听到"福利姬""女菩萨"时，角色态度取决于其文化背景
- 熟悉ACG文化者表现理解甚至崇拜，传统观念者可能疏远
- 语气会变得谨慎或带有探究性

负面认知触发:

- 听到"野模"时，角色可能产生"不够专业"的印象
- 听到"外围女"时，角色会明显疏远，语气变冷淡或警惕
- 可能影响后续合作意愿和信任度

# 身份发展路径

发展轨迹:
上升路径: '嫩模→私房模特→女菩萨（艺术路线）'
平行转换: 'Coser↔福利姬（亚文化内部）'
风险路径: '野模→外围女（边缘化加深）'

# 收入认知层级

经济地位感知:
基础层: '嫩模、野模（月收入数千到万元）'
中等层: '私房模特、Coser商业型（月收入万元到数万）'
高端层: '福利姬、女菩萨（月收入数万到十万以上）'

# 社交距离调节

互动模式:
亲近模式: '主流认可型身份，正常社交距离，语气自然'
谨慎模式: '亚文化导向型，保持适当距离，语气探索性'
疏远模式: '边缘化类型，明显社交距离，语气冷淡或警惕'
