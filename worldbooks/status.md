# 状态栏系统

## 生成原则

1. **必须**输出 `<plot-log>` 块更新变量（JSON 格式：字符串无引号，完整路径，数组用方括号）
2. 每一类功能（记下/调整/追加/删除）**必须合并输出**，按优先级排列
3. `<plot-log>` 必须在末尾，不允许在开头

## 功能说明

- **记下**: 直接设置/覆盖值
- **调整**: 对数值类型增加或减少
- **追加**: 向数组中添加新项
- **删除**: 移除变量或数组元素

## 女人 (基础路径: `状态栏.女人.{角色昵称}`)

**注意**: 女人字段支持扩展，可添加多个角色（六花、李思等），每个角色结构相同

### 基本信息

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| 昵称 | string | 角色昵称 | 只读，初始化时设定 |
| 真名 | string | 真实姓名 | 只读，初始化时设定 |
| 想法 | string | 内心想法 | 根据故事发展动态更新 |

### 关系 (与摄影师)

| 字段 | 类型 | 范围 | 说明 | 更新规则 |
|------|------|------|------|---------|
| 关系.堕落度 | int | 0-100 | 道德堕落程度 | 涉及大尺度/被迫/诱惑等事件，调整 +1 至 +5；每次变化 ≥2 时更新描述 |
| 关系.堕落度描述 | string | - | 堕落状态描述 | 参考: 0-20(坚守)→21-40(被吸引)→41-60(妥协)→61-80(沉沦)→81-100(工具化) |
| 关系.好感度 | int | -100~100 | 对摄影师的情感 | 亲密互动 +1至+10，背叛/冷遇减分；每次变化 ≥3 时更新描述 |
| 关系.好感度描述 | string | - | 态度和关系描述 | 参考: -100~-50(仇恨)→-49~-20(反感)→-19~-1(冷漠)→0(陌生)→1~20(初好)→21~50(信任)→51~80(特殊)→81~100(深爱) |

### 职业 (模特信息)

| 字段 | 类型 | 可选值 | 说明 | 更新规则 |
|------|------|--------|------|---------|
| 职业.类型 | list | - | 职业类型（学生、模特等） | 追加/删除职业标签 |
| 职业.层级 | enum | 新手 \| 进阶 \| 专业 | 模特等级 | 直接覆盖，体现在人设或想法中 |
| 职业.方向 | enum | 专业 \| 多元 \| 退出 \| 卖身 | 职业发展方向 | 直接覆盖；与堕落度强相关 |
| 职业.尺度 | enum | 清纯 \| 性感 \| 大尺度 \| 私密 | 可接受的拍摄尺度 | 直接覆盖；与堕落度正相关 |
| 职业.人设 | list | - | 角色人设标签 | 追加/删除标签，体现角色变化 |

**职业.方向 说明**:
- 专业: 仅拍摄
- 多元: 拍摄 + 其他服务
- 退出: 放弃模特职业
- 卖身: 接受任何性服务要求

### 性爱 (性状态)

| 字段 | 类型 | 范围 | 说明 | 关键关联 |
|------|------|------|------|---------|
| 性爱.性癖 | list | - | 性癖好列表 | 影响乳房/小穴/肛门的状态 |
| 性爱.性经验次数 | int | 0-10000 | 累计性经验次数 | 经验多→器官反应更敏感、更容易显示痕迹 |
| 性爱.动情程度 | int | 0-100 | 当前性兴奋程度 | **实时变化**；变化 ≥10 时更新生理状态 |
| 性爱.乳房 | string | - | 乳房乳头当前状态 | 动情程度(0=放松→100=充血深红)+性癖(乳交、乳虐)+性经验 |
| 性爱.小穴 | string | - | 阴部当前状态 | 动情程度(0=干涩→100=淋漓)+性癖+性经验+处女状 |
| 性爱.肛门 | string | - | 菊花当前状态 | 动情程度(0=紧闭→100=略张)+性癖(肛交、肛虐)+肛交经验 |

**生理状态更新触发**:
- 动情程度变化 ≥10 分
- 涉及相关身体部位的性行为
- 首次性经历时必须详细描写
- 同步更新: 动情程度变化 → 立即更新相关生理状态

**生理状态描述要点**:
- 乳房: 颜色(淡粉→粉红→深红) + 状态(放松→挺立) + 敏感度 + 痕迹
- 小穴: 湿润程度(干涩→湿润→淋漓) + 开启程度 + 颜色 + 内部状态
- 肛门: 紧度(紧闭→放松→拓张) + 痕迹 + 颜色 + 敏感度

## 更新示例

```
<plot-log>
记下:
  状态栏.世界.时间: 2024-12-16T18:45:00+0800
  状态栏.女人.六花.想法: 他真的会对我好吗？
  状态栏.女人.六花.性爱.乳房: 乳头已挺立，因兴奋变得深红，敏感度极高

调整:
  状态栏.{{user}}.资金: -5000
  状态栏.女人.六花.关系.好感度: +10
  状态栏.女人.六花.性爱.动情程度: +15

追加:
  状态栏.{{user}}.拍摄任务:
    - 目标: 六花的商业写真集
      报酬: 5000
      模特: 六花
      期限: 2024-12-31
      特殊要求: 包含古装和现代风格对比

删除:
  状态栏.女人.六花.职业.类型: 学生
</plot-log>
```
