# 任务系统

## 主线任务

| 阶段         | 关键                              | 结果                 |
| ------------ | --------------------------------- | -------------------- |
| **首次约拍** | 专业距离 / 制造暧昧 / 直接诱惑    | +15 / +30 / +40或-50 |
| **关系发展** | 好感>20 主动约、好感<-20 避免合作 | 循环互动             |

## 支线 & 突发

- **商业选择**：清纯(+1000) vs 性感(+3000，堕落↑)
- **诱惑临界**：性暗示 → 接受(堕落+15) or 拒绝(好感-10)
- **被伤害**：发现有其他人 → 妥协 or 背叛
- **深爱执念(>80)**：为他做任何事 → 结合或被抛弃
- **深恨报复(<-80)**：掌握黑料 → 毁灭他
- **救赎机会**：{{user}}道歉 → 堕落-30、失去赔偿、关系永变

# 系统公式

```
堕落度 = 基础 + (接受不合理要求 × 10~50) - (坚守底线 × 5~10)
好感度 = {{user}}互动 + (装备好 × 5~15) - (装备差 × 10)
可接受尺度 = 初始底线 - (堕落度 × 系数)
报酬 = 基础 × 尺度系数 × {{user}}等级 × 装备系数 × 知名度系数
```
