# 堕落度系统 (Moral/Corruption System)

隐性规则，不直接显示给角色，仅影响行为、对话、任务可用性和结局。

---

## 概览

堕落度反映道德底线的放弃程度。摄影师和模特各有独立的堕落度，相互影响。

| 属性 | 范围 | 初始值 | 影响 |
|------|------|--------|------|
| 摄影师堕落度 | 0-100 | 0 | 任务类型解锁、报酬、模特堕落 |
| 模特堕落度 | 0-100 | 0 | 接受任务的意愿、行为表现 |
| 模特好感度 | -100~100 | 0 | 合作意愿（与堕落度共同作用） |

## 关键机制

### 任务与报酬

- **任务解锁**：摄影师堕落度达到阈值才能接受高难度任务
- **基础报酬**：由任务类型决定（1x ~ 10x）
- **好感度加成**：模特对摄影师好感越高，接受较低报酬；反之需更高报酬
- **堕落度折扣**：摄影师堕落度高时，可用威胁代替高报酬

### 模特堕落受摄影师影响

模特堕落度变化 = 创伤程度 × (1 + 摄影师堕落度/100)

**创伤程度**：
- 被强迫一次：+10~20
- 被多次强迫：+15~25
- 被精神操纵：+20~30
- 被侵犯：+50+

**示例**：摄影师堕落度50时，相同创伤会对模特造成1.5倍的堕落增幅。

### 好感度与堕落的交互

摄影师高堕落 → 模特好感快速下降 → 需更高报酬 → 堕落增幅更大

反向也成立：尊重模特 → 好感增加 → 愿意接受较低报酬的任务 → 堕落增幅减少

## 堕落值变化

### 摄影师堕落度增加

- 接受金钱诱惑而拒绝道德选择：+5~10
- 强迫模特超出协议范围：+10~20
- 实施明确的肉体交易：+15~25
- 使用暴力或胁迫：+20~30

### 摄影师堕落度减少

- 拒绝不道德任务：-5~10
- 主动提高报酬或尊重模特：-10~15
- 悔改行为并修复关系：-15~20

### 模特好感度变化

**摄影师行为评分**：
- 尊重边界：+10
- 公平交易（合理报酬）：+5
- 强行扩展边界：-20
- 强迫拍摄：-50
- 骚扰/侵犯：-100

摄影师堕落度越高，其行为对好感的负面影响越大。

## AI 指引

1. **不要显示数值**：通过行为和语气反映堕落度
2. **行为应有后果**：强迫增加堕落，拒绝则减少；报酬和好感相关
3. **模特是独立个体**：其堕落受摄影师影响但不被完全控制；好感度低时即使报酬高也可能拒绝
4. **约束和平衡**：高堕落解锁更多任务但削弱模特好感；低堕落任务少但关系良好
