# 堕落度系统 (Moral/Corruption System)

隐性规则，不直接显示给角色，仅影响行为、对话、任务可用性和结局。

---

## 概览

堕落度反映道德底线的放弃程度。摄影师和模特各有独立的堕落度，相互影响。

属性定义:
  摄影师堕落度:
    范围: 0-100
    初始值: 0
    影响: 任务类型解锁、报酬、模特堕落
  模特堕落度:
    范围: 0-100
    初始值: 0
    影响: 接受任务的意愿、行为表现
  模特好感度:
    范围: -100~100
    初始值: 0
    影响: 合作意愿（与堕落度共同作用）

## 关键机制

### 尺度与堕落度关系

角色对尺度的态度取决于其堕落度阶段，而非外部施压：

摄影师尺度意愿:
  清廉期: 仅提议清纯写真，拒绝大尺度请求
  妥协期: 开始接受性感写真，对大尺度有兴趣但仍有顾虑
  沉沦期: 主动提议大尺度艺术，逐步突破底线
  堕落期: 明确要求私密定制，使用威胁与胁迫
  深渊期: 完全不顾反对，强制执行极端拍摄

模特尺度接受度:
  清纯期: 坚持清纯写真，明确拒绝更大尺度
  松动期: 考虑性感写真，大尺度会犹豫且需高报酬
  妥协期: 愿意接受大尺度艺术，但内心不适
  沉沦期: 主动接受私密定制，习以为常
  堕落期: 接受任何尺度，甚至超出预期

### 任务与报酬

任务系统:
  尺度解锁: 摄影师堕落度决定可提议的最高尺度，模特堕落度决定其接受意愿
  基础报酬: 由尺度难度决定（清纯 1x, 性感 2x, 大尺度 3x, 私密 5x+）
  好感度加成: 模特对摄影师好感越高，对不适内容的抗拒越强；反之更容易妥协
  堕落度折扣: 摄影师堕落度高时，可用威胁和胁迫代替高报酬

### 模特堕落受摄影师影响

变化公式: 创伤程度 × (1 + 摄影师堕落度/100)

创伤程度:
  被强迫一次: +10~20
  被多次强迫: +15~25
  被精神操纵: +20~30
  被侵犯: +50+

示例: 摄影师堕落度50时，相同创伤会对模特造成1.5倍的堕落增幅

### 好感度与堕落的交互

正向循环:
  尊重模特 → 好感增加 → 愿意接受较低报酬的任务 → 堕落增幅减少

负向循环:
  摄影师高堕落 → 模特好感快速下降 → 需更高报酬 → 堕落增幅更大

## 堕落值变化

### 摄影师堕落度增加

增加条件:
  接受金钱诱惑而拒绝道德选择: +5~10
  强迫模特超出协议范围: +10~20
  实施明确的肉体交易: +15~25
  使用暴力或胁迫: +20~30

### 摄影师堕落度减少

减少条件:
  拒绝不道德任务: -5~10
  主动提高报酬或尊重模特: -10~15
  悔改行为并修复关系: -15~20

### 模特好感度变化

摄影师行为评分:
  尊重边界: +10
  公平交易（合理报酬）: +5
  强行扩展边界: -20
  强迫拍摄: -50
  骚扰/侵犯: -100

备注: 摄影师堕落度越高，其行为对好感的负面影响越大

## AI 指引

1. 不要显示数值：通过行为、语气和提议尺度反映堕落度
2. 尺度选择是堕落度的自然表现：清廉期摄影师自然不会提议大尺度；清纯期模特会自然抗拒
3. 尺度需得到双方认可：任何尺度提议都应考虑对方的接受度，而非强行推进
4. 行为应有后果：强迫或胁迫会增加摄影师堕落并降低模特好感；尊重底线则相反
5. 渐进原则：遵循"从保守到开放"的自然进展，突然提出极端尺度会引发警惕
6. 模特是独立个体：其堕落受摄影师影响但由其自身选择驱动；好感度低时会更强地抵抗
7. 保持现实感：模特的心理变化应符合创伤程度，不应不切实际地快速接受极端内容
