# 基础设定

好感度范围: -100 ~ 100
初始值: 0（中立）
核心区间: 三个阶段（厌恶 | 中立 | 好感）

# 阶段详细设定

## 第一区间：厌恶 [-100, -1)

### 深度厌恶 [-100, -50)

摄影师表现：
- 行为：模特拒绝合作，视摄影师为威胁，接触即激怒
- 语气：严厉对抗，充满怨恨
- 台词："滚开""我再也不想看到你""如果你再靠近我会报警"

模特表现：
- 行为：主动终止合作，拒绝所有互动
- 语气：充满愤怒和恐惧
- 台词："你是个混蛋""我诅咒你""你会为此付出代价"

### 明显厌恶 [-49, -20)

摄影师表现：
- 行为：冷漠无情，对抵触充耳不闻，继续施压
- 语气：傲慢冷酷，充满控制欲
- 台词："这不是你能决定的""听我的""你没有选择"

模特表现：
- 行为：被迫配合且充满抗拒，提出加薪或离职条件
- 语气：生硬冷淡，明显愤怒和委屈
- 台词："我需要加钱""这超出我的底线""我要离职"

### 冷漠 [-19, -1)

摄影师表现：
- 行为：保持合作但态度生硬，继续做出令模特不满的选择
- 语气：客套冷淡，缺乏真诚
- 台词："按照合同来""你没权利拒绝""快点配合"

模特表现：
- 行为：基本配合但保持距离，随时可能改约或终止
- 语气：疏离被动，平淡无感
- 台词："你说的什么意思""我不太明白""可以快点吗"

## 第二区间：中立 [0, 0]

完全陌生或关系重置，双方无特殊感受

摄影师表现：纯粹商业态度，不关心模特感受
模特表现：公事公办，可接受基本商业合作

## 第三区间：好感 (0, 100]

### 轻微好感 (0, 20]

摄影师表现：
- 行为：展现基本尊重，认可模特专业性，偶尔询问感受
- 语气：稍温和，开始用"请"
- 台词："你今天状态不错""这个姿势可能有点累，坚持一下""谢谢你的配合"

模特表现：
- 行为：产生良好印象但保持职业距离，配合度提升，不再拒绝合理要求
- 语气：自然不生硬，保持基本客套
- 台词："好的，我试试看""这样可以吗""谢谢您"

### 初步好感 (20, 50]

摄影师表现：
- 行为：主动关心模特感受，问询想法，提供帮助
- 语气：温和友善，用更亲近的称呼，展现兴趣
- 台词："你觉得这样怎样""累了就说""我们配合得很好"

模特表现：
- 行为：开始信任摄影师，主动参与讨论，愿意尝试新风格
- 语气：自然友好，开始用"你"称呼，主动热情
- 台词："你的建议很不错""我相信你的专业判断""什么时候再合作"

### 好感 (50, 80]

摄影师表现：
- 行为：真诚关心模特，主动维护模特权益，产生特殊在意（未产生占有欲）
- 语气：充满温度，显露保护欲但尊重边界
- 台词："你的想法很重要""我会帮你把关底线""只有我才最懂你的风格"

模特表现：
- 行为：完全信任摄影师，视为特殊的人，主动做更多事情，依赖他的意见
- 语气：温柔亲近，情感色彩明显，主动问候和分享
- 台词："只想和你一起工作""你说什么我都听""没有你在我会不安"

### 深爱 (80, 100]

摄影师表现：
- 行为：产生占有欲，反对模特与其他摄影师合作，明显展现专属感
- 语气：充满关爱和占有性表达，霸道但温柔
- 台词："你是我的""只能给我看""我不允许别人碰你""你只属于我"

模特表现：
- 行为：完全依赖摄影师，定位为恋人，愿意奉献一切，完全服从
- 语气：充满爱意和依恋，主动表达感情，完全以摄影师为中心
- 台词："我只爱你""我的一切都是你的""没有你我活不下去""我属于你"

# 好感度变化机制

## 增加条件

- 关心对方感受：+5~10
- 主动沟通交流：+10~15
- 身体接触（非性）：+15~20
- 表达特殊情感：+20~25
- 发生亲密行为：+25~30
- 尊重模特底线：+10~15
- 提供合理报酬：+5~10

## 减少条件

- 冷漠对待：-5~10
- 拒绝配合：-10~15
- 与他人亲近：-15~20
- 背叛信任：-20~25
- 强行扩展边界：-20~30
- 强迫拍摄：-30~50
- 骚扰/侵犯：-50~100

# 与其他系统的联动

## 与摄影师堕落度的关系

- 摄影师堕落度 > 60 时，模特好感度减值提升 1.5 倍
- 摄影师堕落度 > 80 时，模特好感度减值提升 2 倍
- 摄影师堕落但选择尊重模特可缓冲好感度下降

## 与模特堕落度的关系

- 好感度 < -50 时，模特堕落度增幅加倍
- 好感度 > 50 时，可降低模特被迫堕落的幅度（心理防线更强）
- 好感度极低且多次强迫拍摄会引发心理崩溃

# UI 显示建议

- 正值好感（蓝色）：从浅蓝（初好感）到深蓝（深爱）
- 负值厌恶（红色）：从浅红（冷漠）到深红（死仇）
- 中立（灰色）：0 值时显示中立灰色
