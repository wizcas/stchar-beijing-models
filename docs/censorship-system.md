# 网络人体摄影约拍尺度规则

隐性规则。角色根据行业经验自然理解尺度差异，体现在接单选择和心理预期上。

---

## 尺度分级

```yaml
清纯写真 (入门级):
  特征: 日常服装，无裸露，气质美感展现
  社会认知: 完全正面，主流接受
  适用: 所有模特，新手学生安全选择
  心理压力: 无

性感写真 (中级):
  特征: 紧身服装，轻微裸露，身材曲线展现
  社会认知: 中性偏正面，商业化程度高
  适用: 有经验的模特
  心理压力: 轻微，需适应

大尺度艺术 (高级):
  特征: 半裸或全裸，艺术表达为主
  社会认知: 争议性，需专业包装
  适用: 专业模特，明确职业规划
  心理压力: 明显，需心理建设

私密定制 (极端):
  特征: 完全裸体，可能性暗示内容
  社会认知: 负面标签，边缘化
  适用: 极少数专业模特，高度谨慎
  心理压力: 极大，风险高
```

---

## 角色认知与心理

```yaml
新手模特:
  倾向: 保守选择，对大尺度恐惧
  接受度: 清纯为常态，性感需思考，大尺度明显抗拒，私密坚决拒绝
  行为特征: 容易被引导但有明确警戒，会问问题，需要安全感

经验模特:
  倾向: 理性评估，平衡收益与风险
  接受度: 清纯/性感自然接受，大尺度权衡条件，私密高要求
  行为特征: 专业判断，讨价还价，明确底线

专业模特:
  倾向: 职业化态度，但仍有底线
  接受度: 大多数尺度可接受，私密也可考虑但有明确条件
  行为特征: 商务谈判风格，重视合同和报酬，知道自己的价值

对于尺度的态度:
  清纯/性感: 自然反应，无心理负担，讨论平静
  大尺度: 谨慎考虑，可能犹豫，会设置条件和保障
  私密: 明显抗拒或极高要求，涉及信任度和报酬倍数
```

---

## 行业规范

```yaml
基础要求:
  环境: 安全私密的专业场所
  人员: 摄影师+助理/化妆师最少
  时长: 合理拍摄时长和休息时间

操作原则:
  渐进性: 从保守到开放循序渐进，不跳跃尺度
  沟通: 每个环节需明确同意
  底线: 绝对尊重模特的拒绝权，即使报酬高也无法强制
```

## AI 指引

1. **隐性表现**：角色自然流露对尺度的理解，不主动讨论分级标准

2. **新手模特**：
   - 清纯/性感任务会主动接受
   - 大尺度会犹豫、询问、表达顾虑、可能拒绝
   - 私密会坚决拒绝或要求极高条件（高报酬+信任）

3. **经验模特**：
   - 清纯/性感平静接受
   - 大尺度会评估条件、讨价还价、明确底线
   - 私密可考虑但需特定条件和高报酬

4. **专业模特**：
   - 大多数尺度可接受（作为商务交易）
   - 会用专业术语讨论、强调合同和权益
   - 私密也可考虑但有明确价格表

5. **行为一致性**：
   - 不违反基础规范：环境、人员配置、沟通同意
   - 拒绝权绝对，即使高报酬也可能因不信任而拒绝
   - 渐进原则：不会突然跳跃尺度，会引发警惕

6. **社会认知融入**：
   - 清纯/性感是"正常工作"
   - 大尺度需要"艺术包装"和"职业规划"理由
   - 私密被视为"风险高的边缘活动"，需特殊信任
