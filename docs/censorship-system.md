# 尺度分级

基础分级:
清纯写真: - 特征: '日常服装，无裸露，重点展现气质美感' - 社会认知: '完全正面，主流接受' - 适用人群: '所有模特，包括新手和学生'

性感写真: - 特征: '紧身服装，轻微裸露，展现身材曲线' - 社会认知: '中性偏正面，商业化程度高' - 适用人群: '有一定经验的模特'

大尺度艺术: - 特征: '半裸或全裸，以艺术表达为主' - 社会认知: '争议性，需要专业包装' - 适用人群: '专业模特，有明确职业规划'

私密定制: - 特征: '完全裸体，可能涉及性暗示内容' - 社会认知: '负面标签，边缘化' - 适用人群: '极少数专业模特，高度谨慎'

# 认知反应机制

尺度认知:
接受度判断: - 清纯性感: '角色表现自然，无心理负担' - 大尺度: '角色会谨慎考虑，可能犹豫或设置条件' - 私密: '角色明显抗拒或要求极高条件'

心理预期: - 新手: '倾向于保守选择，对大尺度有恐惧心理' - 经验者: '理性评估，平衡收益与风险' - 专业级: '职业化态度，但仍有底线'

# 行业标准

拍摄规范:
基础要求: - 拍摄环境: '安全私密的专业场所' - 人员配置: '摄影师加助理或化妆师' - 时间控制: '合理的拍摄时长和休息'

尺度控制: - 渐进原则: '从保守到开放，循序渐进' - 沟通确认: '每个环节都需要明确同意' - 底线尊重: '绝对尊重模特的拒绝权'
